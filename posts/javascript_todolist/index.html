    因為之前的 Workshop 沒有做出來，就暫時擱在一邊，不過最近要重新練習 JavasScript，所以開始拿之前沒有做出來的 TodoList + LocalStorage 來練手。

    剛開始拿到題目，只看到說要做簡單的新增、刪除、改變 Class，卻沒有先想該怎麼把東西塞進 Cookie 裡面，導致要修改 Code 的時候非常難改，最後選擇砍掉重練。

    這次我改變方式，先從 LocalStorage 下手。

    從 LocalStorage 開始#

    MDN LocalStorage

    看 MDN 可以知道，LocalStorage 只能儲存“字串”，所以存的時候要轉換一下。 這次我把 TodoList 全塞在一個 Array 裡面，用 JSON.stringify() 轉換後，塞到 LocalStorage 裡面就好。

    拿出來的時候，用 for 迴圈或是 map 方法，把 data 一個一個轉出來，加上 Element，然後放在 ul 底下就能完成。

    let data = JSON.parse(localStorage.getItem("todoList")) || [];

const storageData = [...data];

storageData.map((data) => {
  const list = document.createElement("li");
  list.innerHTML = `${data}<span class="close">x</span>`;
  nodeList.appendChild(list);
});

    每次新增或刪除的時候，要重新 set LocalStorage。

    nodeList.addEventListener("click", (e) => {
  if (e.target.className === "close") {
    e.target.offsetParent.remove();
    data = data.filter((item) => item + "x" !== e.path[1].textContent);
    localStorage.setItem("todoList", JSON.stringify(data));
  }
});

if (inputTodo.value.trim() === "") return;
const list = document.createElement("li");

list.textContent = inputTodo.value;
data.push(list.textContent);
localStorage.setItem("todoList", JSON.stringify(data));
list.innerHTML = `${inputTodo.value}<span class="close">x</span>`;
nodeList.appendChild(list);
inputTodo.value = "";
