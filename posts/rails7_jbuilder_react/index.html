Rails 7 使用 jbuilder + React

Posted on:2022年11月13日 |  at 下午02:47

Rails 7 後，直接 new 一個新專案，預設使用 importmap ，而不是 webpack 。

如果想要使用webpack，只要設定帶入 -j webpack；這次要使用的是 jbuilder 指令為 -j jbuilder

$ rails h

// -j, [--javascript=JAVASCRIPT]                             
// Choose JavaScript approach [options: importmap (default), webpack, esbuild, rollup]

這次我選擇直接拔掉 Stimulus.js 和 turbo ，因為不喜歡處理 DOM 。

刪除 Turbo & Stimulus.js

	# Gemfile
	# 註解或刪除 `Gemfile` 裡面的 `stimulus-rails` 和 `turbo-rail`。
	# application.html.erb
	  --	<%= stylesheet_link_tag "application", "data-turbo-track": "reload" %>
    --	<%= javascript_include_tag "application", "data-turbo-track": "reload", defer: true %>
    ++  <%= stylesheet_link_tag "application" %>
    ++  <%= javascript_include_tag "application", defer: true %>

使用 React

安裝 React & React dom

$ yarn add react react-dom

--loader:.js=jsx" 是為了讓它能讀到 jsx 檔案。

# package.json

"scripts": {
  "build": "esbuild app/javascript/*.* --bundle --sourcemap --outdir=app/assets/builds --loader:.js=jsx",
  "watch": "esbuild app/javascript/*.* --watch --bundle --outdir=app/assets/builds --loader:.js=jsx"
}

在 Procfile.dev 加入

js: yarn build --watch

接著，刪除 javascript 資料夾底下的所有檔案，再新增 application.js & components 資料夾

-- javascript
  -- components
     App.js
  application.js

application.js 只當作 entry point 使用，所以東西都會放在 components 或是想再改結構也行

# application.js
import "./components/App"

App.js 裡面就能開始寫 React 了

import React from "react";
import { createRoot } from "react-dom/client";

const root = createRoot(document.getElementById("test"));

root.render(
  <React.StrictMode>
  	<h1>Hello, React！</h1>
  </React.StrictMode>
);

引入寫好的 React 要讓 */html.erb 吃到，只需要 html tag 的 id 和 React 抓到的 id 一樣就好。

<div id="test"></div>

最後，請記得要在 Routes 裡面先設好路徑，再打開 Server foreman s -f Procfile.dev 就能看到 Hello, React! 了。
