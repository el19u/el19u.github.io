    SQL語法複習

    July 21, 2022

    學習 Rails 途中經常透過 Active Record 操作資料庫。ORM 固然方便，不過了解自己寫的 Code 被如何轉換，也是重要的一環，往後也可能碰到，需要自己寫 SQL 語法包進 Scope 裡面，進行更精確的資料庫操作。

    SQL 基本語法#

    在 SQL 語法不一定要大寫，寫成大寫能幫助辨識。

    新增#

    // 第一種寫法： 將 "值" 新增至 "表格" 裡的 "指定欄位"
INSERT table_name(column, column, column)
VALUES (value, value, value);

// 第二種寫法： 將 "值" 依順序新增至 "表格" 裡的 "欄位"
INSERT table_name
VALUES (value, value, value);

    查詢#

    // 查詢 "表格" 的 "欄位"
SELECT column
FROM table;
// 查詢 "表格" 的 "所有欄位"
SELECT *
FROM table;

    查詢不重複資料#

    // 查詢 "表格" 裡 "不重複的" "欄位" 資料
SELECT DISTINCT column
FROM table

    查詢欄位的計算總和#

    // "表格" 裡的 "欄位" 進行加總
SELECT COUNT(column)
FROM table;
